# Inventory Management System - A3 Part1

## Team Information
- **Team Project**
- **Course**: PROG2005 Programming Mobile Systems
- **Assessment**: A3 Part1

## Project Description
This is a cross-platform mobile inventory management system built using Ionic framework and Angular. The application allows users to manage an inventory database while incorporating mobile-friendly UI elements and RESTful API interactions.

## Features
- View all inventory items
- Search items by name
- Add new inventory items
- Edit existing items
- Delete items
- View featured items
- Privacy and Security information page

## API Endpoint
- Server: https://prog2005.it.scu.edu.au/ArtGalley

## RESTful Operations
- GET / - Get all items
- GET /{name} - Get item by name
- POST / - Create new item
- PUT /{name} - Update existing item
- DELETE /{name} - Delete item

## Installation
1. Run `npm install`
2. Run `ionic serve` to start the development server

## Technologies Used
- Ionic Framework
- Angular 17
- TypeScript
